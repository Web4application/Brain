# Quick Start: Brain AI Framework

## 1. Clone the repository
```bash
git clone https://github.com/Web4application/Brain.git
cd Brain
```

## 2. Install dependencies
Python modules:
```bash
pip install -r requirements.txt
```

Arduino setup:
- Install Arduino IDE
- Load code from `Arduino/` folder
- Flash to your device

## 3. Run an example
```bash
python vision/sample.py
```
This should open a demo window or log detection outputs.

## 4. Explore modules
Each module in `/modules` represents a subsystem of the Brain AI architecture.
